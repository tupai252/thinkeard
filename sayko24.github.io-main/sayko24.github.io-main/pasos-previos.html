Thinkeard es un juego de puzzles en el que el objetivo es potenciar la imaginación y la creatividad por el medio de figuras sin una forma final concreta.

Para poder participar en el desarrollo del videojuego necesitaras descargar Unity, que es el motor gráfico que vamos a utilizar. Para ello he creado esta guía que te ayudara a descargar Unity:

  • Visita el sitio web oficial de Unity: unity.com
  • Haz clic en el botón "Download" (Descargar).
  • Descarga e instala el Unity Hub.
  • Abre Unity Hub y crea una cuenta o inicia sesión si ya tienes una.
  • Desde Unity Hub, haz clic en "Install Editor" para instalar la versión de Unity que prefieras.
  • Configura tu primer proyecto y ¡comienza a desarrollar!

    • Se usa Unity como motor grafico y hay archivos C#

    Requisitos del sistema: Nidea.
